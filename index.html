About Me

Hello! I’m Bryony Nicholas, a Digital Marketing BAS student at Bellevue College. I’m passionate about creating visually appealing, user-friendly websites that combine creativity with functionality. This site showcases my class projects and the skills I’ve developed in web design and coding.

Key Concepts I’ve Learned

  • HTML & Semantic Structure: I learned how to create websites using proper HTML tags like <header>, <main>, and <footer> so my pages are well-organized and accessible.
  • CSS Styling & Layout: I gained skills in using CSS for styling, including Flexbox for navigation, creating responsive designs, and applying color schemes that enhance readability.
  • Web Accessibility: I now understand how to make my sites more inclusive by adding alt text to images, maintaining good color contrast, and structuring content for screen readers.
